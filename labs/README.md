# Labs — ITIL 4 Foundation Training (C348)

Eight hands-on labs, worked in class on the **BrightDesk Services** scenario.
Every lab builds on the same organisation, so the course reads as one connected
case — and the ITIL 4 Foundation exam questions you will face test exactly the
concepts you practise here.
