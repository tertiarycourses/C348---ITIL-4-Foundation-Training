# Tools Guide

## Required Tools

| Tool | Purpose |
| --- | --- |
| Markdown editor or word processor | Learner notes, glossary, and reflection answers. |
| Spreadsheet tool | Incident/request classification, change table, service level tracker. |
| diagrams.net | Service value chain, value stream, and stakeholder diagrams. |
| Browser | Access study references and course registration page. |

## Recommended Files

Create these during the course:

```text
itil-glossary.md
stakeholder-map.md
value-stream-map.md
incident-problem-request-table.csv
change-enablement-table.csv
service-level-improvement-plan.md
exam-mistakes-log.md
```

## Evidence Standard

For each lab, capture:

- Scenario interpretation.
- ITIL terms used.
- Decision made.
- Value justification.
- Exam takeaway.
