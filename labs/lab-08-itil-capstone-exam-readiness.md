# Lab 08 - ITIL Capstone and Exam Readiness

## Objectives

- Apply ITIL 4 concepts to one integrated scenario.
- Create a service improvement recommendation.
- Review exam terminology.
- Build a personal exam study plan.

## Scenario

BrightDesk wants to improve the Employee Digital Workplace service. Users complain about slow support, unclear request status, recurring incidents, and poor communication during changes.

## Steps

### 1. Identify the Value Problem


### 2. Map Issues to ITIL Concepts

Create a table:

| Issue | ITIL Concept or Practice | Improvement |
| --- | --- | --- |
| Recurring VPN outage | Problem management | Identify root cause and known error |
| Unclear request status | Service request management | Standard workflow and communication |

Add at least six rows.

### 3. Apply the Seven Guiding Principles

Write one action for each principle. Keep each action practical and scenario-specific.

### 4. Build a Value Stream Summary

Map one service journey from demand to value:

```text

Engage:
Design and transition:
Obtain/build:
Deliver and support:
Improve:

```

### 5. Prepare Exam Mistakes Log

For every missed practice question, record:

```text
Question topic:
Wrong answer chosen:
Correct answer:

```

### 6. Build Final Study Plan

Create a 7-day study plan:

1. Review key concepts and glossary.
2. Review guiding principles.
3. Review four dimensions and SVS.
4. Review service value chain.
5. Review key practices.
6. Complete practice questions.
7. Review mistakes and rest before exam.

## Validation

You should have a value problem statement, issue-to-practice table, guiding principle actions, value stream summary, mistakes log, and study plan.

## Checkpoint Questions

1. Which ITIL concept was hardest to distinguish?
2. What practice appears most often in real service desk work?
3. How does ITIL help teams improve without becoming bureaucratic?
4. What will you review before exam day?

## Exam Focus

The exam rewards precise vocabulary and scenario recognition. Know the definitions, then ask which answer best supports value, collaboration, simplicity, and continual improvement.
